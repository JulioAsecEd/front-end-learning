audio

Usado para embedar conteudo de som para um documento.
contém uma ou mais source tags com diferentes fontes de áudio
texto entre as tags será mostrado somente se o browser não suportar a tag

Atributos:
autoplay: autoplay
controls: controls
loop: loop
muted: muted
preload: auto/metadata/none
src: URL
- Atributos globais
- Eventos globais

source

usada para especificar múltiplos recursos de mídia para elementos de midia, como video, audio e picture

Atributos:
media: media_query
sizes:
src: URL
srcset: URL
type: MIME-type
- Atributos globais
- Eventos globais

track

especifica trechos de texto para audo ou video
Usado especialmente para subtitulos, arquivos de legendas ou outros arquivos contento texto.

Atributos:
default: default
kind: captions/chapters/descriptions/metadata/subtitles
label: text
src: URL
scrlang: language_code
- Atributos globais
- Eventos globais

video

usada para embedar conteúdo de vídeo em um documento

Atributos:
autoplay: autoplay
controls: controls
height: pixels
loop: loop
muted: muted
poster: URL
preload: auto/metadata/none
src: URL
width: pixels
- Atributos globais
- Eventos globais
