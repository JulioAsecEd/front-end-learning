head

É um container para um metadado
Os elementos a seguir são matadados:

  • title
  • style
  • base
  • link
  • meta
  • script
  • noscript

Atributos:
- Atributos globais

meta

Define metadados sobre o documento HTML

Atributos:
charset: character_set
content: text
http-equiv: content-security-policy/ content-type/ default-style- refresh
name: application-name/ author/ description/ generator/ keywords/ viewport - Atributos globais

base

Especifica a URL base e/ou o alvo para todas as URLs relativas em um documento
Deve ter ao menos um href ou um atributo target presete, ou os dois.
Só deve haver 1 elemento no documento.

Atributos:
href: URL
target: _blank/ _parent/ _self/ _top
- Atributos globais
