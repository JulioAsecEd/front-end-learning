img

usada para embedar uma imagem
Imagens não são tecnicamente inseridas numa página, imagens são linkadas para a página
O img cria um espaço para a imagem referenciada.

Atributos:
alt:text
crossorigin: anonymous/use-credentials
height: pixels
ismap: ismap
loading: eager/lazy
longdesc: url
refferpolicy: no-referrer/no-referrer-when-downgrade/ origin/origin-when-cross-origin/same-origin/ strict-origin-when-cross-origin/unsafe-url
sizes: sizes
src: url
srcset: url_list
usemap: #mapname
width: pixels

- Eventos globais

canvas

Usado para desenhar gráficos as pressas, via javascript
É uma tag transparente e é só um container para graficos, deve-se usar um script para desenhar o gráfico.
Qualquer texto dentro do canvas será mostrado em browsers com JavaScript desativado e em browsers que não suportam canvas.

Atributos:
height: pixels
width: pixels
- Atributos globais
- Eventos globais
