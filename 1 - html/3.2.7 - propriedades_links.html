a

define um hyperlink, que é usado para linkar de uma página pra outra

Atributos:
download: filename
href: URL
hreflang: language_code
media: media_query
ping: list_of_URLs
referrerpolicy: no-referrer/no-referrer-hen-downgrade/origin/origin-when-cross-origin/same-origin/strict-origin-when-cross-origin/usafe-url
rel: alternate/author/bookmark/external/help/license/next/nofollow/noreferrer/noopener/prev/search/tag
target: _blank/_parent/_self/_top
type: media_type
- Atributos globais
- Eventos globais

link

Define um relacionamento para o documento atual e um recurso externo.
Mais frequente usada to linkar folhas de estilos ou favicons.
é um elemento vazio, contém atributos somente.

Atributos:
crossorigin: anonymous/ use-credentials
href: URL
hreflang: language_code
referrerpolicy: no-referrer/ no-referrer-hen-downgrade/ origin/ origin-when-cross-origin/ unsafe-url
rel: alternate/author/dns-prefetch/help/icon/license/next/pingback/preconnect/prefetch/preload/prerender /prev/search/stylesheet sizes: HeightxWidth/ any
type: media_type
- Atributos globais
- Eventos globais

nav

Define um conjunto de links de navegação
Note que não são todos os links de um documento que devem estar dentro de um elemento NAV.

Atributos:
- Atributos globais
- Eventos globais
