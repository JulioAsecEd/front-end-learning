select

cria uma lista drop-down
Mais usado para coletar input de usuário
o atributo id é necessário para associar a lista drop-down com uma label

optgroup

usada para agrupar opções relacionadas um elemento select
Se você tiver uma lista longa de opções, grupos de opções relacionadas são fáceis de lidar para um usuário.

option

define uma opção numa lista select
elementos vão dentro de select, optgroup ou datalist
