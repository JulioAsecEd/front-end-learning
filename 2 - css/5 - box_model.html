Box model

O termo box model é usado quando falamos sobre design e layout

O css box model é essencialmente uma caixa que envolve sobre todo elemento HTML. isso consiste em conteúdo, padding, bordas e margens.

  • Content - O conteúdo da caixa, onde o texto e as imagens aparecem.
  • Padding - Limpa uma área em volta do conteúdo. o Padding é transparente.
  • Border - Uma borda que vai em volta do padding e do conteúdo.
  • Limpa uma área fora da borda. Margem é transparente.

o box model possibilita a adicionar borda em volta dos elementos e definir espaço em volta deles.
