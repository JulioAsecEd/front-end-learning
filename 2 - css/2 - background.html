background-image
Especifica uma imagem para usar como fundo de um elemento.

Background repeat
Por padrão a background-image repete uma imagem tanto horizontamente quanto verticalmente.
Algumas imagens devem repetir somente na horizontal ou na vertical, ou parecerão estranhas.
repeat-x - A imagem se repete somente horizontalmente.
repeat-y - A imagem repete somente verticalmente.
no-repeat - A imagem não se repete.

Background-position
Define a posição inicial de uma imagem de fundo
Por padrão ela é top-left e repete tanto verticalmente quanto horizontalmente.
Valores: (se somente um valor for especificado o outro será center)
let top - left center - left bottom - right top - right center - right bottom - center top - center center - center bottom -
x% y% - Vertical/Horizontal
xpos ypos - Vertical/Horizontal basedos em pixels.
initial - Define a propriedade para seu valor padrão.
inherit - Herda do elemento pai.

Background-attachment
Especifica if a imagem de fundo deve rolar ou ser fixada.
Fixed - imagem fixa.
scroll - imagem rola junto com a página.

background-clip
Especifica quão longe o fundo deve se extender dentro de um elemento.
border-box - Valor padrão. O fundo se extende atras da borda.
padding-box - O fundo se extende até a beira interna da borda.
content-box - o fundo se extende até a borda da caixa de conteúdo.

backgroun-origin
Especifica a posição de origen da imagem de fundo.
Não tem efeito se background-attachment estiver fixo.
padding-box - A imagem de fundo começa da borda interna do padding.
border-box - A imagem começa na borda externa.
content-box - A imagem começa dentro do conteúdo.

Background-size
Especifica o tamanho da imagem de fundo.
Existem 4 modos de usar essa propriedade
auto - Valor padrão A imagem fica no seu tamanho original.
length - Define a largura e a altura da imagem de fundo. Se apenas um valor for informado
o segundo fica como auto.
porcentagem - Define o tamanho da imagem e fundo como porcentagem do elemento pai.
cover - Redimensiona a imagem de fundo para cobrir o container inteiro,
mesmo se tiver que esticar a imagem ou cortar uma das extremidades.
contain - redimensiona a imagem de fundo para ter certeza ficará totalmente visível.

shorthand
background:(color)(image)(repeat)(attachment)(position);
