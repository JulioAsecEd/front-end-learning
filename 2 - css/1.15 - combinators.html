Css Combinators

Um seletor CSS pode conter mais de um seletor simples. Entre os seletores simples, nós podemos incluir combinadores.

Existem quatro combinadores diferentes em Css

  • Seletor decendente (espaço)
  • Seletor filho (>)
  • Seletor irmão adjacente (+)
  • Seletor de gêmeos geral (~)

Descendant Selector

Os seletores decendentes correspondem todos os elementos que são decendentes de um elemento especifico.
O exemplo seleciona todos os elementos p dentro de elementos div.
div p

Child Selector (>)

O child selector seleciona todos os elementos que são filhos de um elemento especifico.

Adjacent Sibling Selector (+)

O seletor de irmão adjacente é usado para selecionar um elemento que vem diretamente após de um elemento especifico.
Elementos irmãos devem tem o mesmo elemento pai e "adjacente" significa "seguido imediatamente".
O exemplo seleciona o primeiro elemento p que é posicionado imediatamente depois do elemento div.
div + p

Elemento irmão geral (~)

O seletor de irmão geral seleciona todos os elementos que são irmãos próximos de um elemento especifico.

O exemplo seleciona todos os elementos p que são irmãos próximos de um elemento div
