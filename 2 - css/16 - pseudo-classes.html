pseudo-classes

O que são pseudo classes?

São usadas para definir um estado especial de um elemento.

  • Estilizar um elemento quando um usuário move o mouse sobre.
  • eStilizar links visitados e não visitados.
  • Estilizar um elemento que recebe foco.

Css - first-child

O first-child combina com um elemento especifico que é o primeiro filho de um outro elemento.

Css - lang

Habilita você a definir regras especias para idiomas diferentes.

Pseudo-classes

:checked
Combina todos os input checados(radio e checkboxes) e option.

:disabled
Combina todos os elementos desabilitados(geralmente usado em forms).

:empty
Combina todos os elementos que não tem filhos(incluindo nós de texto).

:enabled
Combina todos os elementos habilitados(geralmente usado em forms).

:first-of-type
Combina cada elemento que é o primeiro filho de um tipo particular.(o mesmo que nth-of-type(1))

:focus
Usado para selecionar um elemento que tem foco. Habilitado em elementos que aceitam eventos de teclado ou inputs de usuários.

:in-range
Seleciona todos os elementos com um valor dentro do de um range especifico.

:invalid
Seleciona elementos de form com um valor que não valida de acordo com as configurações do elemento.

:lang()
Usado para selecionar elementos com um atributo lang com um valor especifico.

:last-child
Combina todos os elementos que são o ultimo filho de seus pais.(igual a nth-last-child(1)).

:last-of-type
Combina todos os elemenots que são o ultimo filgo, de um tipo particular, de seu pai.(o mesmo que nth-last-of-type(1))

:not()
Seleciona todos os elementos que NÃO são o elemento especifico.

nth-child(n)
Combina todo elemento que é o n filho de seu parente.(n pode ser um numero, par ou impar, ou uma formula)

nth-last-child()
Combina cada elemento que é o n filho, indiferente de tipo, de seu pai. Contando do ultimo filho.

nth-last-of-type()
Combina cada elemento que é o n filho, de um tipo particular, de seu pai, contando do ultimo filho.

nth-of-type()
Combina cada elemento que é o n filho do mesmo elemento, de seus pais.

:only-of-type
Combina cada elemento que é o único filho do seu tipo, de seu pai.

:optional
Seleciona elementos de forms que são opcionais.

:out-of-range
Seleciona todos os elementos com valores fora dos ranges especificos.

read-only
Seleciona elementos que são "readonly"

read-write
Seleciona elementos de forms que sao "readable" e "writeable"

:required
Seleciona elementos de forms que são required.

:root
Combina o elemento root do documento.

:target
URLS com um # seguido de uma nome de link para ancora para um certo elemento dentro de um documento.

:valid
Seleciona um elemento de form com um valor que valida de acordo com as configurações do elemento.(funciona somente com elemento de forms com limitações)
