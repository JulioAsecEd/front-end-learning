Attr selectors

É possível estilizar elementos HTML que tem um atributo especifoco ou valores de atributos.

CSS [atributo="valor]Seletor

O seletor [atribute="value"] é usado para selecionar elementos com um atributo especifico e valor.

CSS [atributo~="valor"]

O seletor [atribute~="value"] é usado para selecionar elementos com um atributo contendo uma letra especifica.

CSS [atributo|="valor"]

O seletor [atribute|="value"] é usado para selecionar elementos com um atributo especifico, o que seu valor pode ser um valor especifico exato ou um valor especifico seguido de um hifen(-).

CSS [atributo^="valor"]

O seletor [atribute^="value"] é usado para selecionar elementos com um atributo especifico, o que seu valor começa com um valor especifico.

CSS [atributo$="valor"]

O seletor [atribute$="value"] é usado para selecionar elementos em que seu valor termina com um valor especifico.

CSS [atributo*="valor"]

O seletor [atribute*="value"] é usado para selecionar elementos em que seu valor de atributo contem um valor especifico.
