gradient backgrounds

gradients deixam você exibir transações suaves entre duas ou mais cores especificadas.

css define três tipos de gradientes

  • linear gradient(vai baixo/cima/esquerda/direita/diagonalmente)
  • radial gradient(definido pelo centro)
  • conic gradient(rotaciona sobre o ponto central)

Linear gradient

para criar um gradient linear você deve definir ao menos duas color stops. Color stops são as cores que você vai renderizar transições suaves sobre.

background-image: linear-gradient(to direction, color-stop1, color-stop2, )

usando angulos

se você quer mais controle sobre a direção do gradient, você pode definir um angulo, ao inves de uma direção predefinida.
