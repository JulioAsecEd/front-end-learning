Tabelas

Para especificar bordas use border.

adicione width:100% para que a tabela ocupe a tela toda.

Border collapse

border-collapse define se a borda da tabela deve ser colapsada em uma borda única.(separate/collapse)

vertical-align define o alinhamento vertical(top, bottom, middle)

responsive table

Uma tabela responsiva vai exibir uma scroll bar horizontal se a tela for muito pequena para mostrar o conteúdo completo.
adicione um container com overflow-x:auto em volta da tabela para fazer isso responsivo.

border spacing

border-spacing define a distancia entre as bordas das células adjacentes(só funciona com border-collapse:separated)

caption-side

Especifica o local da legenda da tabela(bottom/top)

emty-cells

Define se serão exibidas bordas em celulas vazias de uma tabela.(show/hide)

table-layout

Define qual algoritmo usado para definir células, colunas e linhas.

  • Auto - Browsers usam um algoritmo de comportamento de tabela automatico. A largura da coluna é definida pelo conteúdo não quebrável mais longo numa célula. O conteúdo vai ditar o layout.
  • Fixed - Define um algoritmo de tabela fixo. A largura da tabela e da coluna são definidos pelas larguras das tabelas e colunas ou pela largura.
