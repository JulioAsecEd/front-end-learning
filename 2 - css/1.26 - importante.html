!important

A regra !important em css é usada para adicionar mais importancia para uma propriedade/valor do que o normal.

In fact, if you see the !important rule, isso vai sobrepor TODOS as regras de estilo anteriores para aquela propriedade especifica num elemento.

Important about !important

A única forma de sobrepor uma regra !important é incluir outra regra !important numa declaração com a mesma(ou maior) especificidade no código fonte, e aqui o problema começa Isso faz o código css confuso e o debbug será dificil, especialmente se você tem uma grande folha de estilo.

May one or two fair uses of !important

Um jeito de usar !important é se você tiver que sobrepor um estilo que não pode ser sobreposto de nenhum outro jeito. Isso pode ser se você está trabalhando em um Content Management System(CMS) e não pode editar o código css. Então você pode definir algum estilo customizado para sobrepor alguns dos CMS styles.
Outro jeito de usar !important é: Assumir que você quer uma aparencia especial para todos os botões numa página.
