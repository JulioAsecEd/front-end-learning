Website Layout

Um website é dividido entre headers, menus, content, e um footer.

header

Um header geralmente é localizado no topo do website(ou logo abaixo de um menu de navegação). Frequentemente contém uma logo ou o nome do website.

navigation bar

Uma barra de navegação contém uma lista de links para ajdar visitantes a navegarem através do website.

content

Possui o conteúdo principal do website. Pode ser dividido em três, duas ou uma parte.

footer

O footer é colocado na parte de baixo da página. Isso geralente contém informações sobre copyright e informações de contato.
