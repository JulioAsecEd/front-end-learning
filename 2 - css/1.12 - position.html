position

A propriedade position especifica o tipe de metodo de posicionamento usado para um elemento(static, relative, fixed, absolute ou sticky).

Existem cinco valores de posição difentes

  • static
  • relativa
  • fixed
  • absolute
  • sticky

static

Elementos HTML são posicionados static por padrão.

Elementos posicionados estáticos não são afetados por propriedades, top, bottom, right, left.

Um elemento com position:static não é posicionado de qualquer modo especial. Isto é sempre posicionado de acordo com o fluxo normal da página.

relative

Um elemento com position:relative é posicionado relativamente a sua posição normal.

Definindo o top, right, bottom, e left propriedades de um elemento relativamente posicionado vai causar com que ele seja ajustado fora de sua posição normal. Outro conteúdo não será ajustado para caber dentro de qualquer gap deixado pelo elemento.

fixed

Um elemento com position:fixed é posicionado relativo ao viewport, que significa que isso sempre estará no mesmo lugar mesmo que a página seja rolada As propriedades top, right, bottom e left são usadas para posicionar o elemento.
Um elemento fixado não deixa um gap na página onde isso normalmente deveria ser alocado.

absolute

Um elemento com position:absolute é posicionado relativo ao proximo ancestral posicionado.
No entando, se um elemento posicionado absoluto não possui ancestrais posicionados ele usa o corpo do documento e se move ao longo do rolamento da pagina

sticky

Um elemento com position:sticky é posicionado baseado no scrool do usuário.

Um elemento sticky alterna entre relative e fixed dependendo da posição do scroll.

clip

Depreciado, useo clip-path. Deixa você especificar um retangulo para clipar um elemento posicionado absoluto. o Retangulo é especificado as 4 coordinadas, todos do canto superior esquerdo ao elemento que vai ser clipado.
