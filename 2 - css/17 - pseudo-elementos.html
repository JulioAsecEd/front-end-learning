pseudo-elements

Pseudo elemento é usado para estilizar partes especificas de um elemento. Por exemplo.

  • Estilizar a primeira letra, ou linha, de um elemento.
  • Iserir conteúdo antes ou depois de um coteúdo ou elemento.

::first-line

Usado para adicionar um estilo especial para a primeira linha de um texto.

Só pode ser aplicado a elementos a level de bloco

::first-line aceita as seguintes propriedades:

  • propriedades de fontes
  • propriedades de cores
  • propriedades de background
  • word spacing
  • letter-spacing
  • text-decoration
  • vertical-align
  • text-transformation
  • line-height
  • clear

::first-letter

Usado para adiconar um estilo especial a primeira letra de um texto.

::first-letter aceita as seguintes propriedades:

  • propriedades de fontes
  • propriedades de cores
  • propriedades de background
  • propriedades de margm
  • propriedades de padding
  • propriedades de borda
  • text-decoration
  • vertical-align(somente se float for none)
  • text-transformation
  • line-height
  • float
  • clear

::before

Usado para inserir algum conteúdo antes de um elemento.

::after

Usado para inserir algum conteúdo depois de um elemento.

::marker

Seleciona os marcadores de uma lista de itens.

::selection

Combina a porção de um elemento que é selecionado por um usuário. Aceita as propriedades color, background, cursos e outline.
