CSS Units

Css tem uma diversos unidades para expressar tamanho.

Algumas propriedades CSS levam valores de "tamanho", como width, margin, padding, font-size, etc.

Tamanho é um numero seguido de uma unidade de tamanho como 10px, 2em, etc...

um espaço em branco não pode aparecer entre um número e uma unidade. No entanto, se o valor é 0, a unidade pode ser omitida.

existem dois tipos de unidades de tamanho: absolutas e relativas.

Tamanhos absolutos

  • cm - centimetros
  • mm - milimetros
  • in - inches(1in = 96px = 2.54cm)
  • px * - pixels(1px = 1/96th de 1in)
  • pt - points(1pt = 1/72 de 1in)
  • pc - picas(1pc = 12pt)

Pixels(px) são relativos a visualização do dispositivo. Para dispositivos low-dpi, 1px é um pixel de dispositivo(dot) wdo display. Para empressões e resoluções altas, 1px implícita multiplos pixels de dispositivo.

Tamanhos relativos

  • em - Relativo ao font-size do elemento(2em significa 2 vezes o tamanho da fonte atual)
