Fontes

Em Css existem 5 familias de fonte genéricas:

  • Serif Fontes tem um pequeno traço nos cantos de cada letra. Elas criam um sentido de formalidade e elegancia.
  • Sans-serif Fontes tem linhas limpas(sem traços pequenos fixados). Eles criam um olhar moderno e minimalista.
  • Monospace Fontes onde todas as letras tem a mesma largura fixa. Eles criam um olhar mecânico.
  • Cursive Fonte imitam a escrita humana.
  • Fantasy Fontes são decorativas/divertidas.

A propriedade Font-Family

A propriedade font-family deve segurar diversas nome de fontes como um sistema de "fallback"

Exemplo

  • font-family: "Times New Roman", Times, serif;
  • font-family: Arial, Helvetica, sans-serif;
  • font-family: "Lucida Console", "Courier New", monospace;

font style

A propriedade possúi tres valores:

  • normal - O texto é mostrado normalmente
  • italic - O texto é mostrado em itálico
  • oblique - O texto está "aprendendo"

Font Weight

Especifica o peso da fonte(normal/bold)

Font Variant

A propriedade font-variant especifica se um texto deve ser mostrado em fonte diminuída. (normal/small-caps)

Font Size

A propriedade font-size define o tamanho do texto.
Pode ser absoluto ou relativo.

Absoluto:

  • Define o texto para um tamanho especifico.
  • Não habilita o usuário a diminuir o tamanho do texto em todos os browsers.
  • Tamanho absoluto é proveitoso quando o tamanho fisico do output é conhecido.

Relativo:

  • Define o tamanho relativo a elementos a volta.
  • Habilita o usuário a mudar o tamanho do texto em browsers.

definir a fonte em pixels dá total controle sobre o tamanho do texto.

Para habilitar o usuário a redimensionar o texto(no menu do browser), uma porrada de desenvolvedores usam em ao inves de pixels.

1em é igual para o tamanho da fonte atual. O tamanho padrão de um texto nos browsers é 16px. Então o tamanho padrão de 1em é 16px.

Font Pairing

Regras do Font Pairing

Aqui estão algumas regras básicas para um bom font pairing:

  • Complemento
    • É sempre seguro achar font pairing que complementem uma as outras
    Uma boa combinação d efontes deve harmonizar, sem ser muito similar ou muito diferente.
  • Use Superfamilia de fotnes
    • Uma superfamilia de fontes é um conjunto de fontes designada a trabalharem bem juntas. Então, usando diferentes Fontes dentro da mesmoa superfamilia é seguro.

    Por exemplo, a superfamilia Lucida contém as seguintes fontes: Lucida Sansa, Lucida Serif, Lucida Typewriter Sans, Lucida Typewriter Serif, e Lucida Math.
  • Contraste é Rei
    • Duas fontes que são muuito similares irão conflitar. Em contrapartida, feito da forma certa, traz o melhor em cada fonte.
    Exemplo: Combinando serif com sans serif é uma boa combinação conhecida.
    Uma superfamilia forte inclúi tanto serif quanto sans serif variações da mesma fonte(e.g. Lucida e Lucida Sans).
  • Escolha apenas um chefe
    • Uma fonte deve ser a chefe. Ela estabelece uma hierarquia para as fontes da sua página. Este pode ser alcançado pela variação do tamanho, peso e cor.
