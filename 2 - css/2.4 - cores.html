Colors

rgba colors

Valores de cor RGBA são uma extensão dos valores RGB com um canal alpha - que especifica a opacidade para uma cor.

Um valor de cor RGBA é especificado com rgba(red, green, blue, alpha). O parâmetro alpha é um número entre 0.0(completamente trasnparente) e 1.0(completamente opaco).

HSL colors

HLS significa hue, saturation e lightness

  1. hue é um grau na roda de cor(de 0 a 360)
    • 0 (ou 360) é vermelho
    • 120 é verde
    • 240 é
  2. Saturação é a porcentagem: 100% é a cor cheia
  3. lightness é também porcentagem: 0% é escuro(preto) e 100% é branco

HSLA colors

um valor HSLA é especificado com: hsla(hue, saturation, lightness, alpha) onde o parametro alpha define a opacidade.

opacity

a propriedade opacity define a opacidade do elemento num todo(tanto cor de fundo e texto serão opacos/transparentes)

a opacity deve ser um número entre 0.0 e 1.0
