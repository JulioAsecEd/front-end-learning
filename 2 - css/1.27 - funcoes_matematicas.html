Math Functions

As funções matemáticas CSS habilitam expressões matemáticas para serem usadas como valores de propriedades.

calc()

A função calc() performa um calculo que pode ser usado como valor de uma propriedade.

max()

a função max() usa o maior valor, de uma lista de valores separados por vírgula, como valor de propriedade.

min()

A função min() usa o menor valor, de uma lista de valores separados por virgula, como valor de uma propriedade.
