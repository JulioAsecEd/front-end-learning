Links

styling links

Links podem ser estilizados diferentemente, dependendo em qual estado eles estão.
Os quatro estados para um link são:

  • a:link - normal, links não visitados
  • a:visited - um link que o usuário visitou
  • a:hover - um link quando o usuário passa o mouse sobre
  • a:active - um link no momento que é clicado

Quando definindo o estilo para diversos estados de link, existem almas regras de ordem

  • a:hover DEVE vir depois de a:link e a:visited.
  • a:active DEVE vir depois de a:hover.
