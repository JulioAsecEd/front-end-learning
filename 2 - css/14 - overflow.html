Overflow

A propriedade overflow controla o que acontece com o conteúdo que é muito grande para caber em uma área.

A propriedade tem os seguintes valores

  • visible - Padrão. O everflow não é clipado. O conteúdo renderiza fora da caixa do elemento.
  • hidden - O overflow é clipado, e o resto do conteúdo ficará invisível.
  • scroll - O overflow é clipado, e um scrollbar é adicionada para visualizar o resto do conteúdo.
  • auto - similar a scroll, mas adiciona uma scrollbar somente quanto necessário.

A propriedade overflow só funciona para elementos de bloco com uma altura especifica.

visible

Por padrão, o overflow é visible, significa que isso não é clipado e renderiza fora da caixa do elemento.

hidden

Com o valor hidden, o overflow é clipado e o resto do conteúdo é escondido.

scroll

Definindo o valor para scroll, o overflow é clipado e uma scrollbar é adicionada para rolar dentro da caixa.
Note que isto vai adicionar uma scrollbar tanto horizontalmente quanto verticalmente(mesmo se você não precisar).

auto

O valor auto é similar a scroll, mas adiciona uma scrollbar somente quando necessário.

overflow-x e overflow-y

O overflow-x e overflow-y especifica se é para trocar o overflow do conteúdo somente horizontalmente ou verticalmente(ou os dois)

overflow-x especifica o quê fazer com os cantos esquedo/direito do conteúdo.

overflow-y especifica o quê fazer com os cantos acima/abaixo do conteúdo.

overflow-wrap

O overflow-wrap especifica se o browser pode quebrar linhas com longas palavras se eles sobrepuserem o container.

valores

  • normal - Palavras longas não vão quebrar, mesmo se elas sobrepuserem o container. Padrão.
  • anywhere - Palavras longas vão quebrar se elas sobrepuserem o container.
  • break-word - Palavras longas vão quebrar se elas sobrepuserem o container.
