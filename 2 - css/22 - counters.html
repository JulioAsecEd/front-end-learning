Counters

Css counters são "variáveis" mantidas pelo CSS que seus valores podem ser incrementados por regras CSS. Counters deixam você ajustar a aparência de um conteúdo baseado no seu posicionamento no documento.

Numeração automática com counters

Css counters são "variáveis". Os valores das variáveis podem ser incrementados por regras CSS.

Para trabalhar com CSS counters vamos usar as seguintes propriedades

  • counter-reset - Cria ou reseta um counter.
  • counter-increment - Incrementa o valor de um counter.
  • Content - Insere conteúdo gerado.
  • counter() ou counters() - adiciona o valor de um counter para um elemento.
