Dropdown

Cria uma caixa dropdown que aparece quando o usuário move o mouse sobre um elemento.

HTMl usa qualquer elemento para abrir o conteúdo do dropdown. Use um elemento de container para criar o conteúdo do dropdown e adicionar o que quiser dentro disso.

envolva um elemento div sobre o elemento para posicionar o conteúdo do dropdown corretamente com o CSS

A classe .dropdown usa position:relative, que é necessário quando nós queremos o conteúdo do dropdown a ser posicionado logo abaixo do botão do dropdown (usando position:absolute).

A classe .dropdown-content contém o real conteúdo do dropdown. Isto é escondido por padrão, e será mostrado em hover.

Se quiser que a largura do conteúdo do dropdown seja larga como o botão do dropdown, defina width 100% e overflow:auto para habilitar scroll em telas pequenas.

Ao inves de usar borda, nós usamos box-shadow para fazer o menu do dropdown parecer como um "cartão".
