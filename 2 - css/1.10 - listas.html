Listas

A propriedade lista do CSS permite você:

  • Definir diferentes marcadores de itens para uma lista ordenda.
  • Definir diferentes marcadores de itens para uma lista não ordenda.
  • Definir uma imagem como marcador de item.
  • Adicionar cores de fundo para listas e itens.

list-style-image

Especifica uma imagem como o marcador de item da lista.

list-style-type

Especifica o tipo de marcador de item da lista.

position list item mark

list-style-position especifica a posição do marcador do item da lista.(inside/outside)

Shorthand

list-style:(list-style-type)(list-style-position)(list-style-image)
